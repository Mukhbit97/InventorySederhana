# Sistem Inventaris Sederhana

Proyek ini adalah sistem manajemen inventaris barang sederhana yang dibuat dengan Python.

## Fitur
- Menambah, melihat, mengupdate, dan menghapus barang (CRUD).
- Menjual barang dan mengurangi stok secara otomatis.
- Data disimpan secara persisten menggunakan file JSON.
- Menggunakan konsep Pemrograman Berorientasi Objek (OOP).

## Cara Menjalankan
1. Pastikan Python 3 terinstal.
2. Buat lingkungan virtual: `python -m venv venv`
3. Aktifkan lingkungan: `source venv/bin/activate`
4. Install dependensi: `pip install -r requirements.txt`
5. Jalankan program: `python main.py`
